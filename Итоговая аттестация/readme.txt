Для запуска ноутбука "Итоговая аттестация Пристов М.А.ipynb" необходимо распаковать два архива с данными, данные
находятся по ссылке (https://drive.google.com/uc?export=download&id=10ov5WIxq6ocXp852JSDLcPIv6UULq9Uc)
в переменную path указать путь до тренеровочного набора данных, в переменную target указать путь до файла
"y_test.csv", который находится в архиве "Данные для предсказания".
Для предсказания в разделе "Загрузка и обработка данных для предсказания" необходимо указать путь до папки "test",
находящийся в архиве "Данные для предсказания".  В переменную pred указать путь до файла"y_predict",
который находится в архиве "Данные для предсказания".
Чтобы не обучать заново модели, в папке model находятся все обученые модели, которые необходимо подгрузить в код.
